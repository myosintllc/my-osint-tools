What is a Bookmarklet?

"A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands that add new features to the browser. They are stored as the URL of a bookmark in a web browser or as a hyperlink on a web page. Bookmarklets are usually small snippets of JavaScript executed when user clicks on them. When clicked, bookmarklets can perform a wide variety of operations, such as running a search query from selected text or extracting data from a table." - Wikipedia

How do I use these tools?

These bookmarks live in your browser (or you can use some of them right on this page!). You use them in certain situations.If you want them in your browser's bookmarks bar, you have 2 ways of adding them:

⭐Easy Method⭐
  1. Find the bookmarklet you want
  2. Click and drag the bookmarklet's name (left column in table above) it into your bookmarks bar.
  3. Click that bookmark to use the tool. ⚠You will need to allow popups in your browser!⚠
  4. Done!
🔨The "More Manual Method"🔧

Click to expand

  1. Visit https://github.com/myosintllc/my-osint-tools/tree/main/
  2. Open the bookmarklet you wish to use.
  3. Scroll down to the // Copy and paste the data below section.
  4. Copy the text starting with javascript: and ending with ()
  5. Go to your favorite browser and make a new bookmark. We like doing this in our Bookmarks Bar but you can do it wherever.
  6. Name it whatever you wish.
  7. In the URL section, paste the code your copied. You need to paste everything from the javascript: all the way to the ending ().
  8. Save the bookmark/bookmarklet.
  9. Now you are set to run or use the bookmarklet. Some you need to be on a certain web page to use (like the Facebook ID one needs to be on a Facebook page like https://www.facebook.com/zuck). Navigate to the correct page or, for those that do not need a certain page, just click the bookmark.
  10. WARNING: Some of these bookmarklets open one or more browser tabs. You may need to ALLOW POP-UPS in your web browser to get those tabs to show.

Common Issues and Problems (FAQ)

Bookmarklet not working correctly for you? Here are some helpful steps you can take. Just click the button below that is closest to your issue.

❓ Nothing happened when I used bookmarklet
  • Ensure that popup windows are allowed in the current browser tab.
  • Try reloading the page and then using the bookmarklet.
⚡ I got an error when running the bookmarklet
  • Write down the error and research it on Google.
  • Try reloading the page and then using the bookmarklet.
😇 I'd like to have you put one of my bookmarklets in here.
  • Well aren't you nice! Please submit an issue in our GitHub repository. When you create a new issue, you will be prompted for what we need to review your bookmarklet. We will then send you an email response.
